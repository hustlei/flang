## 控制语句

### 条件

#### if语句

单选结构

```
if(条件表达式){条件为true时执行的语句}
```

双选结构

```
if(条件表达式)
  {条件为true时执行的语句}
else
  {条件为false时执行的语句}
```

多选结构

```
if(条件表达式1){语句}
elif(条件表达式2){语句}
...
elif(条件表达式n){语句}
else{语句}
```

### 循环

for循环

~~~
for 变量 in 列表{
 循环体
}
~~~

while循环

~~~
while(循环条件表达式){ // 循环体 } 
~~~

跳出循环体用break。

提前结束本轮循环用continue。
