# 常量、变量

## 常量、变量定义

~~~
//常量，编译期求值，运行期间不能改变
const PI=3.14159  //自动推断类型
const a:Int = 25

//单赋值变量，初始化后值将不能改变
let x:Int = 5
let line = sum(1,2)  //自动推断类型

//变量
var b:Str = "abc"
var x = [1, 2, 3]  //自动推断类型
~~~

连续赋值

~~~
const a,b,c = 1,2,3
const l,m:Str = "x", "y"

var x, y = 1.0, "str" 
var i:Int, s:Str = 1, "1"
~~~

## 动态类型变量

为了在某些特殊情况下更加灵活，支持动态类型。动态类型编译时不检查类型，运行时尝试调用相关函数。

~~~
dynamic obj = getApp()
dynamic child obj.getChild()
var dynName = child.name()
~~~

> 一般情况下不建议使用dynamic类型
>
> 在与动态语言交互会很方便。
> 或者调用activex/COM对象，类似office，autocad等应用，
> 后绑定运行对象的方法时，采用dynamic类型会很方便。
